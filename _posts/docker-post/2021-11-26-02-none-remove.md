Dockerfile을 통해 이미지를 생성하거나,   
다른 여러 이유들 때문에 `<none>`이라는 이미지가 생성될 때가 있습니다.

해당 이미지는 아래의 명령어로 삭제할 수 있습니다.

```
$ docker rmi $(docker images -f "dangling=true" -q)
```
